  • "an oil paiting of a cat curled up by a fireplace"
  • "a pencil drawing of a tutle with a patterned shell"
  • "a photo of a playful puppy running on the beach"

When the number of ingerence steps increases, the quality of the generatedd images improves significantly and follows the text prompts more closely. With only 20 inference steps, the images lack detail. For example, in the first image of the puppy on the beach, the puppy's features are not very distinct and lacks fur and the sand looks untouched which is unrealistic if the puppy was running on it. In contrast, with 500 inference steps, the puppy's fur texture is more defined, and the sand shows signs of disturbance from the puppy's movement, making the scene more believable. Similarly, the cat by the fireplace image and the dog on the grass image both show significant improvements in detail and visualizes the prompt more accurately of the actions and settings described.

The random seed I set for this part was 100.

Part A.1.6 Classifier-Free Guidance

To improve the image quality from a noisy image, I implemented classifier-free guidance by computing a conditional and an unconditional noise estimate. In my iterative_denoice_cfg function, I implemented a similar process to iterative_denoise from Part A.1.4, but in each iteration, I computed both the conditional and unconditional noise estimates using the pretrained UNet model. Then, I combined the two estimates using a guidance scale factor to get a final noise estimate. Finally, I used the final noise estimate to compute the less noisy image at the next timestep using the same formula from Part A.1.4.

Part A.1.8 Visual Anagrams

To implement Visual Anagrams and create optical illusions with diffusion models, I created a function visual_anagrams that takes in two text prompts and a random noisy image. Following a similar process to iterative_denoise_cfg from Part A.1.6, in each iteration, I computed two conditional noise estimates using the pretrained UNet model for both text prompts on the original image and the flipped image. Then, I found the less noisy image for the both and flipped the flipped image back before combining them into an average image.

Below are illusions and their corresponding text prompts:

"an oil painting of an old man" + "an oil painting of people around a campfire"

"a photo of a bunny sitting on a windowsill" + "a photo of a dog"

"a pencil drawing of a dachshund" + "a pencil drawing of a turtle with a patterned shell"

Part A.1.9 Hybrid Images

To create hybrid images, I created a function make_hybrids that takes in two text prompts and a random noisy image. Following a similar process to iterative_denoise_cfg from Part A.1.6, in each iteration, I computed two conditional noise estimates using the pretrained UNet model for both text prompts. Then, I put one image through a high-pass filter and the other image through a low-pass filter before combining them into a single hybrid image.
